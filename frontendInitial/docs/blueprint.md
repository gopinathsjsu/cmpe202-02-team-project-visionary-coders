# **App Name**: Market Square

## Core Features:

- Item Listing: Allow users to list items for sale with details, photos, and pricing.
- Search and Filter: Enable users to search for items based on keywords, categories, location, and price range.
- Location Services: Identify user location to show nearby listings and allow filtering by distance.
- Direct Messaging: Facilitate direct communication between buyers and sellers within the app.
- Image Analyzer: AI-powered tool that will attempt to auto-categorize new listing based on their uploaded images, suggest reasonable prices based on similar postings, and to analyze for fake or harmful product details.
- User Profiles: Enable users to create and manage their profiles with options to view their listings and messages.

## Style Guidelines:

- Primary color: HSL 210, 67%, 53% (RGB hex: #3498DB) for a clean and trustworthy feel.
- Background color: HSL 210, 20%, 96% (RGB hex: #F0F4F7) for a soft and unobtrusive backdrop.
- Accent color: HSL 180, 55%, 40% (RGB hex: #34A898) to draw attention to important interactions.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, providing a modern and readable interface.
- Use clear and recognizable icons to represent categories, actions, and features.
- Implement a grid-based layout for organized display of listings.
- Incorporate subtle animations for transitions and interactions to enhance user engagement.
